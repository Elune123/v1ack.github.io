FAQ

  • Difference between Docs View and PDFview

    Docs View is a newer version of PDFview

    PDFview have problems on latest Tizen version with sending files, in some cases phone can't connect to watch

    Docs View supports not only PDF, but also markdown and txt files (in UTF-8 encoding)

    You can try both demos apps and decide what is better for you

  • Why Docs View is another app

    I'm not able to update the app in Galaxy Store because of changes for latest Tizen version

  • I'm already bought PDFview. What can i do?

    If you don't have problems with sending files - use PDFview

    In other cases you can install Docs View demo for sending files and open them from old paid app

    Also i'm going to make a sale for some time after release Docs View

  • What is demo

    It's an app with limited use

    You can open only 3 different files. It's very cool to try app before paying and to find out how it works
